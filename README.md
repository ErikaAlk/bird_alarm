# 鸟瘾闹钟

用真实鸟鸣叫醒你的闹钟。内置十余种鸟鸣，支持从 xeno-canto 搜索下载海量鸟声，或导入本地音频文件作为铃声。

> **关于这个 fork**：本仓库是 [Eric6286/bird_alarm](https://github.com/Eric6286/bird_alarm) 基于**个人使用习惯**对原项目的修改版。所有改动仅出于自用需要（去掉强制认鸟挑战、锁屏直接关闹钟、深色模式、闹钟 Live Updates 等），**与原作者无关**，原作者不对这些改动负责。

## 功能

- **鸟鸣闹钟**：从内置鸟声库或 xeno-canto 选择铃声
- **锁屏直接关闹钟**：响铃时在锁屏页直接显示「正在叫的是哪只鸟」+ 关闭 / 贪睡 5 分钟，无需解锁进 App（已移除原版的强制认鸟挑战）
- **鸟声库管理**：搜索、下载、导入本地音频，支持 IOC 鸟类名录检索
- **多组闹钟**：支持设置多个闹钟并独立管理

## 重要：必须允许后台运行

**闹钟必须保持 App 在后台持续运行才能正常响铃。**

请务必关闭系统对本 App 的后台自动清理：

### Android
1. 进入「设置」→「电池」（或「省电管理」）
2. 找到「鸟瘾闹钟」→ 选择「不限制」或「允许后台活动」
3. 关闭「自动清理后台应用」中对本 App 的限制

不同手机品牌路径略有差异：
- **小米 / Redmi**：设置 → 省电与电池 → 应用省电 → 鸟瘾闹钟 → 无限制
- **华为 / 荣耀**：设置 → 电池 → 启动管理 → 鸟瘾闹钟 → 手动管理，全部开启
- **OPPO / 一加**：设置 → 电池 → 省电模式 → 应用耗电管理 → 鸟瘾闹钟 → 允许
- **vivo**：设置 → 电池 → 后台耗电管理 → 鸟瘾闹钟 → 允许后台运行
- **三星**：设置 → 电池 → 后台使用限制 → 鸟瘾闹钟 → 不受限制

### iOS
iOS 系统限制后台应用运行，建议将 App 保留在前台（锁屏前不要完全退出），或在锁屏前确认闹钟已设置成功。

## 使用 xeno-canto 搜索鸟声

在鸟声库中点击搜索，输入鸟种名称（中文、英文或学名均可）即可从 xeno-canto 获取录音列表。如需更高请求额度，可在设置中填入个人 xeno-canto API Key。

## 更新记录

### v1.1.0（2026-06-21）

> 以下改动均基于个人使用习惯，与原作者无关。

- **修复**：下载鸟鸣时整个 App 卡死——把原生音频转码从平台主线程移到后台线程执行，下载/转码全程界面保持可滑动、可操作。
- **改动**：移除「强制认鸟挑战」玩法。响铃时锁屏页直接显示「正在叫的是哪只鸟」，并提供「关闭闹钟」与「贪睡 5 分钟」按钮，**无需解锁手机**即可关闭。
- **改动**：亮屏且已解锁时，响铃改用通知（heads-up）提醒，不再强行全屏接管；锁屏 / 息屏时才拉起全屏响铃页唤醒。
- **修复**：`AlarmReceiver` 与 `AlarmSoundService` 共用通知 id 1001 互相覆盖，导致「停止」按钮有时不出现——改为响铃通知统一由前台服务负责。
- **新增**：响铃前 10 分钟弹出倒计时通知，可直接在通知里点「关闭闹钟」取消这次响铃，无需进 App。在 Android 16（API 36）上会提级为 **Live Update**（状态栏胶囊 / 锁屏 / 息屏常驻）；更低版本上显示为普通常驻通知。
- **升级**：`compileSdk` / `targetSdk` 升到 36（Android 16），并声明 `POST_PROMOTED_NOTIFICATIONS` 权限，使上面的通知能被系统提级为 Live Update。**构建前请确保本机已安装 Android SDK Platform 36 且 Android Gradle Plugin ≥ 8.6**，否则构建会报缺少平台。
